# Crypto
Java based encryption / decryption program, with GUI interface

## Features:
   * AES Encryption
   * Simple GUI Interface
   * Save/Load/Generate Keys
   * Encrypt Text/Images/Docs/Videos And Many more!
   * Hide Text within any image!
